# Aula 01 – Apresentação da Disciplina e Introdução à Programação Web

## 📌 Apresentação da Disciplina

A disciplina tem como objetivo apresentar os fundamentos da Programação Web, combinando princípios de design visual com tecnologias web para criação de interfaces digitais funcionais e atraentes.

O professor responsável possui formação em:
- Mestrado em Ciência da Computação
- Especialização em Inteligência Artificial (em andamento)
- Graduação em Análise e Desenvolvimento de Sistemas
- Experiência como Gerente de Projetos, Professor Universitário e Coordenador Acadêmico

## 🧩 Conteúdo Programático

### 1️⃣ Versionamento e Deploy
- Conceito de controle de versões
- Uso de Git, GitHub, GitLab e Bitbucket
- Deploy e ambientes de desenvolvimento
- Integração Contínua (CI/CD)
- Automação e boas práticas

### 2️⃣ Arquitetura de um Site
- Definição de escopo
- UX/UI
- Prototipação
- Validação e iteração de ideias

### 3️⃣ JavaScript (Vanilla JS)
- Linguagem interpretada essencial no front-end
- Manipulação do DOM
- Comunicação com servidores
- Base para frameworks modernos

### 4️⃣ Servidores, Hospedagem e CMS
- Conceito de servidores
- Website builders
- Criação de sites sem codificação intensiva
- Vantagens e limitações dessas ferramentas

### 5️⃣ Frameworks Front-end
- React, Vue e Angular
- Componentização
- Gerenciamento de estado
- Rotas e integração com APIs

### 6️⃣ Frameworks Back-end
- Conceitos e arquitetura (MVC, REST)
- Integração com banco de dados
- Autenticação e segurança

### 7️⃣ Banco de Dados NoSQL
- Modelo não relacional
- Schema flexível
- Escalabilidade horizontal

## 📊 Critérios de Avaliação

Fórmula:

(P1 × 0.25) + (P2 × 0.25) + ((PJ + AT) × 0.25)

Onde:
- P1 = Prova 1
- P2 = Prova 2
- PJ = Projeto
- AT = Atividades

---

## 📝 Atividades Iniciais

1. Formar grupos de 3 a 5 integrantes.
2. Definir proposta de projeto.
3. Criar repositório no GitHub.
4. Criar arquivo `.md` com resumo da Aula 01.
5. Analisar 3 sites:
   - Identificar tecnologias utilizadas.
   - Criar representação visual no Miro.
   - Salvar imagem no repositório.
